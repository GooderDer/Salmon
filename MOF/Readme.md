# 基于 CrystalNN 的 MOF 开放金属位识别程序

## 一、项目简介

本项目用于识别金属有机框架材料（MOF）中的开放金属位（Open Metal Site, OMS）。

程序基于 `pymatgen` 中的 CrystalNN 算法计算金属原子的配位环境，通过配位数与理想配位数对比，自动筛选潜在的低配位金属中心，从而识别可能的开放位点。

本项目可用于：

* MOF 吸附性能前筛选
* 催化活性位点分析
* 高通量材料筛选
* 论文数据复现

---

* `src/`：开放位识别算法实现

---

## 三、算法原理

开放金属位通常表现为：

* 金属中心存在低配位状态
* 配位数小于其常见理想配位数

本程序流程如下：

1. 读取 CIF 结构文件
2. 使用 CrystalNN 计算金属原子的配位邻居
3. 统计配位数（Coordination Number, CN）
4. 与预设理想配位数进行比较
5. 标记低配位金属为开放位候选

判断逻辑：

```
若 CN < 理想配位数 → 判定为开放位候选
```

## 五、使用方法

示例运行方式：

```
python src/main.py data/xxx.cif
```

程序输出内容包括：

* 金属原子索引
* 元素类型
* 配位数
* 是否为开放位候选
